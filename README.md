### 코딩 스크립트

```javascript
// Express 라이브러를 씁니다
const express = require("express")

// express 프레임워크를 사용해서 앱을 정의 합니다:
const app = express()

// Public 폴더 안에 저희가 필요한 웹사이트 파일이 있다고 말합니다
app.use(express.static("public"))

```
